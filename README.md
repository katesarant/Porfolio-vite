# 🚀 Vue 3 Portfolio

A personal portfolio app built with **Vue 3**, **Tailwind CSS**, and **Axios**. It features modular components, dynamic content, and mock API integration via **JSON Server**.

## 🔍 Features
- Vue 3 + Composition API
- Tailwind CSS styling
- Axios for API calls
- Skills & Experience from JSON
- Contact form with toast notifications

## 🛠 Tech Stack
- **Vue 3**, **Vue Router**
- **Tailwind CSS**, **PrimeVue**, **PrimeIcons**
- **Axios**, **JSON Server**
- **Vue Toastification**

## 📁 Structure
```
src/
├─ components/
├─ views/
├─ App.vue
└─ main.js
```

## ⚙️ Setup

```bash
git clone https://github.com/YOUR_USERNAME/vue3-portfolio.git
cd vue3-portfolio
npm install
npm run dev
```

### (Optional) Mock API
Make sure you have:
- `portfolio.json`
- `skills.json`
- `db.json`

Run mock API:

```bash
npm run api
```
